2. Client-Side Security - Payment Bypass

Description: Premium course access is controlled client-side only.

Vulnerability: The /api/user/paid-status endpoint returns payment status, but frontend trusts it without server verification.

How to exploit: Intercept /api/user/paid-status response ‚Üí Change "paid": false to "paid": true ‚Üí Unlock all premium course videos

  1. Set up Burp Suite or use browser DevTools Network tab

  3. Have admin visit the admin panel - this will automatically make requests to hidden API endpoints
  4. Intercept/capture the requests to see what API endpoints admin is calling:
    • GET /api/admin/settings

  8. Success! You can access admin data because the endpoint only checks authentication, not admin role!

Example: Using cURL or Burp Repeater, send:
